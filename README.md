## 🚀 프로젝트 소개

이 프로젝트는 도서관의 대출 및 반납 절차를 디지털화하여, 사용자들이 더 편리하게 도서를 검색, 예약, 대출, 반납할 수 있도록 지원하는 백엔드 시스템입니다. RESTful API와 데이터베이스를 기반으로 구축되어, 사용자와 관리자 모두가 효율적으로 서비스를 사용할 수 있는 환경을 제공합니다.

## 🌟 주요 기능

### 1. User
- 회원가입
- 로그인
- 비밀번호 초기화 요청
- 비밀번호 초기화

### 2. Book
- 전체 도서 조회
- 개별 도서 조회
  
### 3. Like
- 좋아요 추가
- 좋아요 삭제

### 4. Cart
- 장바구니 담기
- 장바구니 아이템 목록 조회
- 장바구니 도서 삭제

### 5. Order
- 주문하기
- 주문 목록 조회
- 주문 상세 조회

### 6. Category
- 카테고리 전체 목록 조회
